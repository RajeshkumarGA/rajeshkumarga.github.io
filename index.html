Efficient and Secure Content Processing and Distribution by Cooperative Intermediaries

Efficient and Secure Content Processing and Distribution by Cooperative Intermediaries

AIM

To develop a java based application that can implement efficient and secure content processing across data transmission.

OBJECTIVES

•To implement Parallel Secure content service (PSCS) protocol for caching and analyzing the cooperative intermediates

•To introduce key management scheme across the intermediate nodes

•To maintain a profile table of the intermediate nodes and a centralized key storage server

•To compare the performance of the developed system with the centralized approach

•To develop a java based application that can demonstrate the proposed system practically.

CONCLUSION

In this project, a solution has been presented for secure content services that are characterized by robust and scalable network architecture. Client is being permitted by protocol to confirm that the data received is authentic and the data transformations are authorized properly. During the transmission process, the confidentiality of data is assured. In order to enhance the performance of system and also to support parallel content services, distribution of load is highlighted by means of P-proxies. Several applications can be deployed easily, since the modification is not needed to the existing content distribution systems. Additionally, the main approach here is extensible i.e. towards new environment our architecture can be adapted easily whenever a new content service is needed.
